# CWA-CALCULATOR
The CWA Calculator is a high-end, responsive web application specifically designed for Biomedical Engineering (BME) students at KNUST (Class of 2029). It provides a seamless interface for students to calculate their Cumulative Weighted Average (CWA) and Grade Point Average (GPA) using the exact KNUST credit-weighted formula. 
📊 CWA Calculator — Asante Inc.
A premium, client-side CWA and GPA estimation tool built for KNUST Biomedical Engineering (BME1) students.  

🌟 Key Features
Official BME1 Integration: Pre-loaded with Year 1, Semester 1 courses (Algebra, Cell Biology, Applied Electricity, etc.) and their respective credit hours.  

Target CWA Engine: Reverse-calculate the scores you need in remaining subjects to hit a specific grade class (e.g., First Class).  

Dual-Scale Support: Toggle instantly between CWA (/100) and GPA (4.0) scales.  

Interactive Modes:

iPhone 15 Pro: Experience the calculator inside a virtual smartphone with a live Dynamic Island update.  

Bloomberg Terminal: A high-contrast, "physical terminal" aesthetic for data-focused calculations.  

Privacy First: No data is stored, shared, or transmitted. All calculations happen locally in your browser session.  

Customizable: Add, edit, or remove courses to adapt the tool for future semesters or different departments.  

🛠️ Technical Stack
Frontend: Semantic HTML5, CSS3 (with custom "Grain" textures and CSS variables for theming).  

Logic: Vanilla JavaScript for real-time calculations and DOM manipulation.  

Theming: Integrated Dark/Light mode and mobile-optimized responsive design.  

📂 How to Use
Login: Enter your name to personalize the session.  

Input Scores: Enter your marks out of 100 for each course.  

AI 150 Toggle: Use the dedicated switch to include or exclude the "Responsible AI for All" course from your weighted total.  

Target Setting: Use the "Target CWA" tab to see what scores are required for your desired final average.  

Export: Download a .txt report of your projected results for your records.
